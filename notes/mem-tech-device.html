Mem Tech Device

February 18, 2019


Memory Tech Devices tech device is a feature that nobody that works on x86 has ever heard of. According to their own definition "We're working on a generic Linux subsystem for memory devices, especially Flash devices.". Seems like their goal is to provide an abstraction over a bunch of different memory techs. eg. NAND, NOR flash etc.. It should be noted, that block devices like MMC, eMMC, SD etc do not work with this driver.
